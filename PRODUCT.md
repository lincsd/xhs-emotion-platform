# 🌸 小红书情感账号·运营管理平台

> 免费在线工具，无需下载安装，手机电脑均可用
> 🔗 **https://lincsd.github.io/xhs-emotion-platform/**

---

## ✨ 这是什么？

一个专门为小红书运营者打造的 **AI 全自动内容管理平台**。

输入情感分类，点一个按钮，AI 自动：
1. 联网搜索小红书当前爆款笔记
2. 分析流量密码和标题规律
3. 生成高质量内容 + 封面图 + 配图

全程无需手动写一个字。

---

## 🚀 核心功能

### 📊 数据看板
- 笔记总数、草稿、已发布、总收入一览
- 浏览量 / 点赞 / 收藏汇总统计
- 近期笔记动态

### ✨ AI 内容生成
- **联网搜索爆款**：调用 Google 搜索实时抓取小红书热门笔记数据
- **智能分析**：提炼标题规律、内容结构、标签组合、流量密码
- **一键生成**：基于分析结果生成 1~10 条高质量情感笔记
- **AI 配图生成**：自动生成封面图 + 配图（Nano Banana 模型）
- **A/B 测试**：同一主题生成多个版本对比
- 支持自定义 Prompt 精细调控风格

### 📝 笔记管理
- 草稿箱、已发布、已排期分类管理
- 编辑 / 删除 / 状态变更
- 笔记预览（手机端样式还原）

### 📅 发布日历
- 可视化排期，查看每天发布计划
- 推荐最佳发布时间（7 个时段评分）
- 批量排期功能

### 💰 收入追踪
- 记录广告费、佣金、直播等各类收入
- 月度收入统计图表
- 变现方案推荐（接广、带货、知识付费等）

### 🛠️ 智能工具箱
| 工具 | 说明 |
|------|------|
| 🚫 违禁词检测 | 发布前扫描敏感词，降低被限流风险 |
| 💯 笔记评分 | 多维度质量评估（标题、正文、标签、封面） |
| 🔥 热门话题 | 追踪当前赛道热点关键词 |
| 🏷️ 标签引擎 | AI 智能推荐精准标签组合 |
| 🔄 笔记改写 | 多模式智能改写，同内容多版本 |
| 💬 评论话术库 | 热评回复模板 + AI 生成话术 |
| 🔍 竞品分析 | 对标分析同类账号内容策略 |

### 🎀 账号管理
- **账号包装**：AI 生成账号名称、简介、头像风格
- **账号数据**：粉丝增长、互动率等数据记录
- **多账号管理**：同时管理多个运营账号，数据互不干扰
- **模板库**：保存常用高效模板
- **数据导出**：一键导出所有笔记和数据

## 🤖 技术说明

- AI 生成基于 **Gemini 2.5 Flash** 模型（Google 最新）
- 联网搜索基于 **Google Search Grounding**（实时数据）
- 图片生成基于 **Imagen / Nano Banana** 模型
- 后端部署在 **Render**（新加坡节点），数据持久化存储
- 前端托管在 **GitHub Pages**，全球 CDN 加速
